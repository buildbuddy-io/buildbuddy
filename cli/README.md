# BuildBuddy CLI

Install BuildBuddy CLI
```
curl -fsSL install.buildbuddy.io | bash
```

Kick off a Bazel build using BuildBuddy UI
```
bb build //...
```

Kick off a Bazel build using BuildBuddy Cache
```
bb build //... --cache
```

Kick off a Bazel build using BuildBuddy RBE
```
bb build //... --remote
```
